About Me



My name is Ankit Muneshwar and I’m a Unity Game Developer from India. I got into contact with my first video game, Super Mario Bros.Later I became more obsessed with games right away. That was my first glimpse into game development and when I tried creating my own games for the first time. It wasn’t until a few years later that I realised I didn’t have much talent for Game Art or Game Design, Around that time I learned about Unity Game Engine and found out that there was quite an active one revolving around one of my favorite games. After some time I became more and more experienced with it and even started writing more codes on Unity. I only learned C, Java programming at school, I enjoyed programming so much that I eventually started learning C# by myself and looking into game programming. I decided to try programming a game called Super Mario Bros. 2D and succeeded at it. At that time I had already made my decision to aim for a career as a Game Programmer, I dove into C# and gained a lot more experience with object-oriented programming and game development, which I used to program some more advanced 2D and 3D games using Unity. Above all really helped me to find myself and come to the conclusion that Game Programming is truly one of my biggest talents and that I love being a part of game development more than anything else.
