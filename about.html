About Me



My name is Ankit Muneshwar and I’m a Unity Game Programmer from India. I got into contact with my first video game, Super Mario Bros. in my childhood.Later I became more obsessed with games right away and from that moment on spent most of my free time with them.That was my first glimpse into game development and when I tried creating my own games for the first time. It wasn’t until a few years later that I realised I didn’t have much talent for Game Art or Game Design, but rather for the more technical aspects of game development. Around that time I learned about Unity Game Engine and found out that there was quite an active one revolving around one of my favorite games. After some time I became more and more experienced with it and even started writing more codes on Unity. In 2013 I enrolled for Computer Science at school and started taking my first actual programming lessons. While I only learned C, Java programming at school, I enjoyed programming so much that I eventually started learning C# by myself and looking into game programming. During my final year at school I had to work on a team project in Computer Science. As a team, we decided to try programming a game called Super Mario Bros. 2D and succeeded at it. At that time I had already made my decision to aim for a career as a Game Programmer, but didn’t quite know how to achieve that goal yet. After graduating from school I immediately applied for Game Programming course on coursera. Before even beginning , I dove into C# and gained a lot more experience with object-oriented programming and game development, which I used to program some more advanced 2D and 3D games using Unity and learned that I enjoy Gameplay Programming even more than Engine Programming. Above all really helped me to find myself and come to the conclusion that Game Programming (and especially Gameplay Programming) is truly one of my biggest talents and that I love being a part of game development more than anything else.
